# PRODIGY_ML_05

## Food Recognition and Calorie Estimation

This project detects food items from images using Deep Learning and estimates calorie values.

## Features
- Detects food from images
- Shows prediction confidence
- Estimates calories
- Supports multiple food images

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNetV2
- NumPy
- PIL

## How to Run

```bash
python food_recognition.py
```

## Sample Foods Tested
- Pizza
- Burger
- Banana
- Sandwich
- Panipuri
- Kabab
